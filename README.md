# Sudoku CSP Solver

## Description
This project implements a Sudoku solver using Constraint Satisfaction Problem (CSP) techniques.

## Techniques Used
- Backtracking Search
- Forward Checking
- AC-3 Algorithm

## Input Format
- 9x9 Sudoku grid
- 0 represents empty cells

## Files Included
- sudoku_solver.py → main solver
- easy.txt → easy puzzle
- medium.txt → medium puzzle
- hard.txt → hard puzzle

## Output
The program prints:
- Solved Sudoku board
- Number of backtrack calls
- Number of failures

## How to Run
```bash
python sudoku_solver.py
